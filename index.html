LightCMS

LightCMS is a lightweight, minimalist CMS that makes creating and updating your content simple.

Features

Streamlined, intuitive interface

Focus on your content - we'll take care of the rest.

Lightweight, minimalist principles

Small footprint with a believe in less is more mentality.

Markdown support

Use markdown to quickly and easily write content.

Quickly create and edit posts

No admin pages or unnecessary routing to create and edit content.

Easy to get started

Follow our steps on Github to get started.

Open source

Check us out on Github to contribute!
